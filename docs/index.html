AnimeShin

AnimeShin is a modern unofficial AniList companion client. Track your anime & manga, manage your library, and optionally open media using user-configured extensions.

Visit the Repository Back to Hub

Key Features

AniList Sync

Full AniList integration for anime & manga tracking.

Russian Titles

Displays localized titles via Shikimori metadata.

Extensions

Supports optional user-configured extensions connecting to external services.

Media Player

Built-in player capable of opening media provided by configured external services.
